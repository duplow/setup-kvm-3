# Módulo de vídeo Displayport

Este módulo faz chaveamento de 3 monitores para 1 usando conectores displayport 1.4+

## Características

- Tipo de conectores: Displayport 1.4+
- Tensão de operação: 3.3V
- Consumo em operação: 60mA
- Corrente máxima na saída: 500mA

## Recursos

- Switch por I/O ✔️
- 4K@30 ✔️
- HDP ❌
- Audio ❌
