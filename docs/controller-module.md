# Módulo Controlador

Este módulo faz o chaveamento das entradas e saídas e controla os demais módulos.

## Características

- Tipo de conectores: N/A
- Tensão de operação: 3.3V

## Recursos

- Switch por botão físico ✔️
- Switch por I/O ✔️
- Switch por controle remoto (IR) ❌
- Switch por teclado ❌
